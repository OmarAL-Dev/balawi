# Balawi

**A modern, lightweight, and blazing-fast desktop Kanban board application.**

---

## About the Project
**Balawi** (an Arabic term playfully referring to overwhelming tasks or "burdens") is a productivity tool designed to help you organize your daily workflow into a simple, visual Kanban interface. 

Built with **Tauri**, the application delivers native desktop performance with minimal resource consumption. It places a strong emphasis on providing a highly polished, native **RTL (Right-to-Left)** layout specifically tailored for Arabic users, breaking away from the typical "mirrored afterthought" approach seen in many apps.

### 🚧 Project Status: Work in Progress (WIP)
> **Note:** This project is currently in its **early development phase**. Core features, UI/UX refinements, and core functionalities are still being actively built and tested.

---

## Current Features ✨
- 🖥️ **Native Desktop Experience:** Runs independently as a desktop app, offering superior speed compared to traditional web wrappers.
- 🌍 **Flawless RTL Support:** A thoughtfully designed Right-to-Left interface optimized for Arabic text flow.
- 🎨 **Modern Dark Theme:** A sleek, eye-friendly dark mode designed to reduce eye strain and maintain focus.
- 🔡 **Premium Typography:** Integrates the **IBM Plex Sans Arabic** font to ensure crisp, highly readable Arabic diacritics and text.
- ⚡ **Ultra-Lightweight:** Powered by the Tauri engine, ensuring near-zero RAM bloat compared to Electron-based alternatives.

---

## Tech Stack 🛠️
- **Framework:** [Tauri v2](https://tauri.app/) (Rust + Webview)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Typography:** IBM Plex Sans Arabic Font
- **Environment:** Built and tested on Fedora Linux
